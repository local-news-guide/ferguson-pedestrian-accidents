# Ferguson & Ferguson — Personal Injury Law Firm Demo

Premium law firm demo site built with vanilla HTML, CSS, and JavaScript. Inspired by the design language of high-end personal injury firms.

## Live Preview

Open `index.html` in any browser — no build step or server needed.

## Design System

- **Typography:** Playfair Display (headings) + Inter (body)
- **Color palette:** Deep navy (#0c2340) + gold (#c19a3e) + warm neutrals
- **Components:** Hero with image overlay, practice area cards, attorney profiles, case results, testimonials, FAQ accordion, multi-column footer
- **Interactions:** Scroll-triggered reveal animations, sticky header with shrink, mobile slide-out nav, dropdown menus, counter animation
- **Responsive:** Desktop, tablet, mobile breakpoints at 1024px, 900px, 600px, 400px

## Pages

- **index.html** — Full-service firm landing page (hero, stats, about, practice areas, case results, attorneys, testimonials, process, FAQ, CTA, footer)
- **pages/** — 12 geo-targeted city pages for Cullman County pedestrian accidents:
  - Cullman, Hanceville, Good Hope, Baileyton, Berlin, Colony
  - Dodge City, Fairview, Garden City, Holly Pond, South Vinemont, West Point

## Structure

```
├── index.html          Landing page
├── css/style.css       Design system v2.0
├── js/main.js          Navigation, scroll animations, interactions
├── pages/              12 city-specific service pages
├── images/             (placeholder for local images)
└── README.md
```

## Deploy to GitHub Pages

1. Go to **Settings > Pages** in the GitHub repo
2. Set source to **Deploy from a branch** > **master** > **/ (root)**
3. Site will be live at `https://local-news-guide.github.io/ferguson-pedestrian-accidents/`
